# Crypto Price Tracker Web3

This project helps beginners learn how crypto apps show live prices and connect with blockchain tools.

You will learn:
- How to fetch crypto prices
- How to use a public API
- How to connect Web3 provider
- How to show data on a website
- How real crypto dashboards work

## Requirements
- Node.js installed
- Basic JavaScript knowledge
- Internet connection

## Setup

1. Clone the repo
git clone https://github.com/yourusername/crypto-price-tracker-web3.git

2. Install packages
npm install

3. Run the app
npm start

4. Open browser
http://localhost:3000

## Project Structure

- index.html = Main webpage
- app.js = Frontend logic
- api.js = Price API handler
- web3.js = Blockchain connection
- config.js = Settings
- server.js = Local server
- package.json = Project info
- utils.js = Helper functions
- logger.js = Logs
- .env.example = Environment example

## Features
- Show Bitcoin price
- Show Ethereum price
- Refresh button
- Connect wallet option
- Simple Web3 example

## Learning Goal
This project shows how Web2 APIs and Web3 tools can work together in one app.
